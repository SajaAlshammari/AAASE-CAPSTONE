# Limitations

- Resume extraction currently supports PDF files only.
- Employee account provisioning is simulated and does not connect with real enterprise systems.
- SQLite is used as a lightweight database for demonstration purposes.
- The system currently provides backend APIs only without a user interface.
- Generated employee credentials are for demonstration purposes only.
